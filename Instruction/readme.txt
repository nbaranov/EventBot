Для работы бота необходимо установить Python 3 и несколько его библиотек. Так же нужен браузер Google Chrome (для линукс систем Firefox)

1) Установка Python
1.1) Переходим на сайт https://www.python.org/downloads/windows/ и скачиваем Latest Python 3 Release - Python 3
    (На момент написания это Latest Python 3 Release - Python 3.8.2)
1.2) Во время установки обязательно установите флажок на «Add Python X.Y to PATH» в мастере настройке. (вроде как прямо на первом окне установщика)

2) Установка библиотек.
2.1) Открываем командную строку, лучше с правами администратора. и перейти в папку с программой.
2.2) Ввести в командную строку:
    pip install -r requirements.txt
    (в линуксе использовать pip3)
    или поочередно вставить команды
    pip install requests
    pip install beautifulsoup4
    pip install selenium
    pip install lxml
2.3 Чтобы не вводить логин и пароль от портала при каждом запуске можно создать переменные среды, для этого пишем в консоль:
set NOK_PORTAL_LOG=login
set NOK_PORTAL_PAS=password
(скорее всего чтобы переменные применились в среду нужно будет перелогиниться)


3) После установки
3.1) Открываем файл selenEvents.py для редактирования любым редактором.
3.2) Находим словарь executor_list и оставляем там исполнителей, с которыми будет работать бот. Чтобы не удалять лишних можно закоментировать символом #
3.3) Чтобы уменьшить количество телодвижений бота, лучше настроить фильтр ивентов как указано на картинке EventsFilter.png

4) Запуск.
4.1) Я предпочитаю запускать его на отдельном рабочем столе, чтобы сильно не мешался, для это жмем Win+Ctrl+D
4.2) Открываем папку с ботом, кликаем по selenEvents.py
4.3) Откроется консоль с просьбой ввести логин от портала, после еще один запрос на пароль (пароль скрыт, он не отображается при наборе, но он набирается) вводим и нажимаем Enter.
4.4) После чего бот начнет работу с ивентами.
